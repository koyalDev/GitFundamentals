1. Install Homebrew:

3.Configure Git:

```shell
git config --global user.name "John Doe"
git config --global user.email johndoe@example.com
```

4. Setup Sublime as Global editor:

```shell
git config --global core.editor "subl -w"
```

5. Install Git:

```shell
brew install git
```
This installs git and autocompletion for git.
